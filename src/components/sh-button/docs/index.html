button

Usage

        
          <sh-button>
            <p>
              Button
            </p>
          </sh-button>

Reference

Attributes

Attribute Type Description
variant String
  • Style variant.
  • Defaults to 'primary'.

Slots

Slot Description
default The text content of the button.
before-icon Material Icon that comes before the text.
active-before-icon Material Icon that comes before the text when the 'active' prop has been toggled true.
after-icon Material Icon that comes after the text.
active-after-icon Material Icon that comes after the text when the 'active' prop has been toggled true.

Events

Event Payload Type Description
click String
  • Fires when the button is clicked.
