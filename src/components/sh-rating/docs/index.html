Rating

Usage

      
          <sh-rating level="3.2" count="72"></sh-rating>

Reference

Attributes

Attribute Type Description
count Number Adds a number in parenthesis after the rating.
level Number Accepts tenth values from 0 - 5.0, for example: 4.3
choose Boolean Adds hover/click handlers for selecting a rating.

Events

Event Payload Type Description
rating-chosen Number Emits with the payload of the value that was selected — usually 1 - 5.

Examples

With [count]

With [choose]
