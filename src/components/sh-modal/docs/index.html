modal

Usage

      
          <sh-modal header="Modal Header" id="uniqueId">
            <p>
              Lorem ipsum doler.
            </p>
          </sh-modal>

Reference

Attributes

Attribute Type Description
variant String
  • Change the way the modal is rendered.
  • Options: primary (default), secondary, tertiary
header String Content for the modal header.
footer String Content for modal footer.

Slots

Slot Description
header HTML content for the header of the modal.
content HTML content for the main area of the modal.
footer HTML content for the footer of the modal.

Public Methods

Function Params Description
open
  • none
Trigger the modal. This will close other open modals.

Events

Event Payload Type Description
modal-opening String
  • Fires before the modal opens.
  • Payload is the ID of the modal that will open.
modal-opened String
  • Fires immediately after the modal opens.
  • Payload is the ID of the modal that has opened.
