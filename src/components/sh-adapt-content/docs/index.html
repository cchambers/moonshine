Adapt Content

Usage

This component allows you to perform and manage element level media queries... for example, if you needed the text to change when the sh-adapt-content container was 400px wide:

      
          <sh-adapt-content>
            full desktop content
            

              small
            

            

              medium
            

            

              large
            

            

              xtra-large
            

          </sh-adapt-content>
