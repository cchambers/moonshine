app banner

Usage

        
          <app-banner days-after-close="3" days-after-open="30"></app-banner>

The App Banner loads when the configured conditions are met.

Reference

Attributes

Attribute Type Description
days-after-close Number
  • The amount of time (in days) before the banner will show again after it has been closed.
days-after-open Number
  • The amount of time (in days) before the banner will show again after 'View' has been clicked.

Slots

Public Methods

Function Params Description
open
  • none
Trigger the app banner.
close
  • none
Close the app banner.

Examples

Basic

Implementations would not include the [active] attribute by default.
