OO Classes

Usage

OO classes that can be used to style or space content.

        
          <p class="px30 font-primary" >Lorem ipsum dolor...</p>

Animation

Class Effect
.stage-left Position something off-screen to the left.
.stage-right Position something off-screen to the right.

Box Model

Class Effect
.pad{-edge}{-amount}
  • Modify the padding of an element. Left+right edge combined is x, top+bottom edges combined is y.
.stage-right Position something off-screen to the right.
