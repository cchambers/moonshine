OO Classes

Usage

OO classes that can be used to style or space content. They are not used in component creation, but the variables used to create them are. 

      
        <p class="px30 font-primary" >Lorem ipsum dolor...</p>

Animation

Class Effect
.stage-left Position something off-screen to the left.
.stage-right Position something off-screen to the right.

Box Model

Class Effect
.pad{-edge}{-amount}
  • Modify the padding of an element. Left+right edge combined is x, top+bottom edges combined is y.
.margin{-edge}{-amount}
  • Modify the margin of an element.
  • Left+right edge combined is x, top+bottom edges combined is y.
  • Options for 'amount' are: none, 'quarter', 'half', and 'double'.

Typography

Alignment

Class Effect
.text-right Right align text.
.text-left Left align text.
.text-center Center align text.
.text-justify Align text justified.
.force-justify Align text justified forced (justifies the last line and single-line paragraphs).
