Component Prototype

Usage

This is the base object of every component. Be sure to check out Event Bus for details on event handling.

Reference

Attributes

Attribute Type Description
hidden Boolean Hide from view.
variant String
  • Different variants generally provide different functionalities or layouts.
  • Defaults to 'default'. Options are usually named in ordinal fashion: primary/secondary/tertiary/quaternary/quinary/etc

Properties

Attribute Type Description
uuid String Unique ID for this instance of a component.

Events

Event Payload Type Description
component-ready VueInstance Emitted when component is mounted.
